## 🚀 Quick Deploy (5 Minutes)

### 1️⃣ Prepare for GitHub
```bash
# In your project directory:
git init
git add .
git commit -m "Initial commit: Fertilizer Prediction System"
```

### 2️⃣ Push to GitHub
```bash
# Go to https://github.com/new and create a new repo
# Then run (replace with your username/repo):
git remote add origin https://github.com/YOUR_USERNAME/fertilizer-prediction.git
git branch -M main
git push -u origin main
```

### 3️⃣ Deploy on Render
- Go to **render.com** (free account)
- Click **New Web Service**
- Select your GitHub repository
- Render auto-detects `Procfile` and deploys automatically!

### 4️⃣ Share Your App
After 2-5 minutes, your app is live at:
```
https://your-unique-name.onrender.com
```

## 💡 Example: Make a Prediction via API

```bash
curl -X POST https://your-app.onrender.com/predict \
  -H "Content-Type: application/json" \
  -d '{
    "Temparature": 30,
    "Humidity": 66,
    "Moisture": 38,
    "Soil Type": "Red",
    "Crop Type": "Millets",
    "Nitrogen": 23,
    "Potassium": 0,
    "Phosphorous": 24
  }'
```

**Response:**
```json
{
  "prediction": "28-28",
  "confidence": 0.72,
  "probabilities": {
    "10-26-26": 0.03,
    "14-35-14": 0.05,
    "28-28": 0.72,
    ...
  }
}
```
